Design

1. Reorganization of the product arquitecture

Based on all the feedback and information gathered, I worked on the creation of a new arichitecture for the product. The objective was to create something aligned with the users mindset, so that it would make sense for them where to find each thing. Also it focused on creating a user flow that enabled maximum optimization depending on the way the users used the platform.

2. Creation of low definition prototypes

Created low definition prototypes to define well the user flows, together with some UI decisions. First, I worked on pen and paper, to experiment many different paths easily and effectively. Then I moved on to creating the wireframes, first ver basics, and the more detailed in order to be able to test them.
